# FRS On-Air

Eine moderne Next.js-Anwendung für das Freie Radio für Stuttgart (FRS 99.2 MHz). Die App bietet Live-Streaming, eine interaktive Programmübersicht, ein durchsuchbares Archiv vergangener Sendungen (Mediathek), Profile der Radiomacher\*innen sowie eine Liste kommender Events. KI-Funktionen werden durch Genkit bereitgestellt.

## Features

*   **Live-Streaming**: Höre das Programm des FRS in Echtzeit.
*   **Programmübersicht**: Entdecke das Tagesprogramm.
*   **Mediathek**: Greife auf das Archiv vergangener Sendungen zu.
*   **Macher\*innen-Profile**: Lerne die Menschen hinter den Sendungen kennen.
*   **Termine**: Bleibe über kommende Events auf dem Laufenden.
*   **KI-gestützte Empfehlungen**: Erhalte personalisierte Vorschläge für Sendungen und DJs.

## Tech Stack

*   **Framework**: Next.js (App Router)
*   **Sprache**: TypeScript
*   **UI**: React, ShadCN UI, Tailwind CSS
*   **Generative AI**: Google Genkit
*   **Audio Player**: Zustand
